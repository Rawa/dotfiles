# dotfiles 

## Targets
- `common` - Common files for all setups
- `laptop` - laptop specific configuration files 
- `server` - server specific configuration files

## Install
    git clone --recursive https://github.com/Rawa/dotfiles

## Usage
    stow --target=~/ <TARGET_FOLDER>
